Denny's DAO


This is a DAO (Decentralized Autonomous Organization). It allows you to create a decentralized, democratically run organization. A DAO uses peer-to-peer communication for information sharing. The management and processing of information (adding blog posts, updating a forum thread, changing a nickname, etc.) is done through the Ethereum client. The Ethereum client makes sure that everyone's local data is synchronized with the rest of the network. It also makes sure that all changes to the data are done in accordance with the agreed-upon rules.

DAO's can have many different rules. They do not even have to be democratically run. Rules are updated and kept current by the same mechanisms that processes the other information.
